# Ethelred

> [!NOTE]
> Following a decision by the tech leadership group,
> Ethelred was decommissioned in June 2026.
> * The repository was archived, the open issues were closed, and the open PRs were closed.
> * The Slack channels were archived.
> * The App Platform app and Spaces Object Storage bucket were deleted.
> * <https://ethelred.opensafely.org> was redirected to <https://github.com/opensafely-core/ethelred>.
> * This callout was added.
> * This decision was communicated to the Bennett Institute, on Slack.

[Ethelred][1] is an Old English personal name, meaning "noble counsel" or "well-advised".
This Ethelred hopes to avoid the fate of [its namesake][2],
who was infamously not ready.

This Ethelred is:

* several tasks for extracting, transforming, and loading data from multiple, heterogeneous sources.
  You will find them in `tasks`.
* a [Streamlit][] app for representing that data to facilitate more effective decision-making.
  You will find it in `app`.
